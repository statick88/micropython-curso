🤖 Pybricks vs LEGO Spike: Guía de Orientación

🤖 Pybricks vs LEGO Spike: Guía de Orientación

¿Por qué este anexo?

Este anexo existe porque hay dos caminos para programar tu robot LEGO:

  1. LEGO Spike App (oficial): Entorno visual de bloques
  2. Pybricks (alternativo): Código Python real

Buena noticia: ¡Puedes usar ambos! No son excluyentes.

🎯 Quick: ¿Cuál debería usar?

Situación Recomendación
Estudiante muy pequeño (6-8 años) LEGO Spike App
Principiante sin experiencia previa LEGO Spike App → Pybricks
Ya sabe algo de programación Pybricks directamente
Quiere aprender Python “de verdad” Pybricks (este curso)
Aula con presupuesto limitado Pybricks (más opciones)
Competiciones FIRST LEGO League Pybricks (más control)

🔄 El Flujo de Transición: Spike → Pybricks

Muchos docentes prefieren empezar con Spike (más visual) y luego migrar a Pybricks (más poderoso):

    ┌──────────────────┐
    │   LEGO SPIKE    │ ← Empezar aquí (más accesible)
    │   APP (bloques)  │
    └────────┬─────────┘
             │
             ▼ Transición natural
    ┌────────▼─────────┐
    │     PYBRICKS     │ ← Código Python real
    │  (MicroPython)    │
    └──────────────────┘

¿Qué conceptos se transfieren?

Concepto en Spike Equivalente en Pybricks
Bloque “Encender motor” motor.start()
Bloque “Esperar” wait() o sleep()
Bloque “Repetir” for o while
Bloque “Si…entonces” if/else
Variables Variables (exactamente igual)

Diferencias Clave

Spike (bloques) Pybricks (código)
Arrastras bloques Escribes texto
Los errores son visuales Los errores son mensajes de texto
Difícil copiar código Easy copiar y pegar
Limitado a lo que existen bloques Puedes crear cualquier cosa

📊 Ventajas de Pybricks sobre LEGO Spike

1. Código es Transferible

# Este código sirve para CUALQUIERA que aprenda Pybricks
# No importa si tienes Spike Essential, Prime, o Technic Hub
def mover_cuadrado(lado_cm):
    for i in range(4):
        robot.straight(lado_cm)
        robot.turn(90)

2. Comunidad Activa

  • Foros de Pybricks: Respuestas rápidas
  • GitHub: Código fuente disponible
  • Ejemplos: Miles de proyectos compartidos
  • Documentación: Completa y actualizada

Sintaxis de programación real ❌ ✅
Preparación para career tech ❌ ✅

🚀 Cómo Empezar con Pybricks (Este Curso)

Este curso está diseñado para ensenar directamente con Pybricks, sin pasar por Spike.

Si ya conoces Spike

  1. No necesitas desaprender nada - Los conceptos son los mismos
  2. Solo aprenderás la “traducción” de bloques a código
  3. Ganarás superpoderes: más control, más flexibilidad

Si nunca has usado nada

  1. Perfecto: Empiezas con la herramienta más poderosa desde el inicio
  2. No tienes que “migrar” después
  3. Aprendes Python de verdad
