🎯 ¿Por qué usar un simulador?

Beneficio Descripción
💰 Ahorro Sin necesidad de comprar hardware para cada estudiante
🔧 Debug Ejecutar código paso a paso, ver variables
👥 Accesibilidad Todos pueden practicar desde su computadora
🛡️ Seguridad No hay riesgo de dañar componentes físicos

📊 Comparativa de Simuladores

Simulador Tipo Costo Pybricks Ideal para
Pybricks Web Online Gratis ✅ Principiantes
Thonny + REPL Local Gratis ❌ MicroPython puro
ESP-32 Simulator Hardware ~$15 ❌ Prototipado real
ViperIDE Online Gratis ✅ Clases remotas

📝 Código Compatible con Simuladores

Este código funciona EN TODOS los simuladores y en hardware real:

# Código universal - funciona en Pybricks Web, Thonny, y hardware real
from pybricks.hubs import PrimeHub
from pybricks.parameters import Icon
from pybricks.tools import wait

# Inicializar Hub
hub = PrimeHub()

# Mostrar icono
hub.display.show_icon(Icon.HEART)
wait(1000)

# Reproducir sonido
hub.speaker.beep(frequency=440, duration=500)

# Apagar
hub.display.off()

print("¡Código ejecutado exitosamente!")

🎓 Ejercicio: Probar en Simulador

Objetivo

Ejecutar este código en Pybricks Web y verificar que funciona.

Código de prueba

from pybricks.hubs import PrimeHub
from pybricks.tools import wait

hub = PrimeHub()

# Secuencia de LEDs
for i in range(5):
    matriz = [[0 for _ in range(5)] for _ in range(5)]
    matriz[i][i] = 100  # Diagonal
    hub.display.show(matriz)
    wait(200)

hub.display.scroll("OK!", wait=True)

Pasos

  1. Abrir https://code.pybricks.com
  2. Copiar el código de arriba
  3. Click en ▶️ (Run)
  4. Verificar que los LEDs se encienden en diagonal
