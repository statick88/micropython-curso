Unidad 1: Tu Primer Programa

Una situación del día a día

Imagina que tienes un robot en tu aula y quieres que cuando presiones un botón, el robot diga “¡Bienvenidos!”.

¿Cómo le explicarías al robot lo que tiene que hacer?

En tu mente podrías pensar: 1. “Cuando alguien presione el botón” 2. “Que muestre el mensaje ‘¡Bienvenidos!’” 3. “Que encienda una luz verde”

Eso exactamente es lo que vamos a programar. Vamos paso a paso.

Primero: Instalar las herramientas

Antes de dar instrucciones al robot, necesitamos las herramientas correctas. Usaremos solo software gratuito.

Paso 1: Instalar Thonny

Thonny es un programa donde escribiremos las instrucciones para el robot. Es como un cuaderno de trabajo [1].

Si usas Windows: 1. Ve a https://thonny.org 2. Descarga la versión para Windows 3. Ejecuta el archivo y sigue las instrucciones (siguiente, siguiente, siguiente…)

Si usas Mac: 1. Ve a https://thonny.org 2. Descarga la versión para Mac 3. Abre el archivo .dmg y arrastra Thonny a Aplicaciones

Si usas Linux: Abre una terminal y escribe:

sudo apt install thonny
Note¿Por qué Thonny?

Thonny fue creado específicamente para personas que están aprendiendo a programar. Tiene todo integrado: editor, forma de ejecutar código, y herramientas de depuración.

Y es completamente gratis.

Paso 2: Instalar el software en el robot

El robot (LEGO Technic Large Hub) necesita un programa especial para entender Python. Se llama Pybricks [2].

  1. Conecta tu robot a la computadora usando el cable USB-C
  2. Presiona el botón de encendidodel robot (el botón central)
  3. En Thonny, ve a: Tools → Options → Interpreter
  4. Selecciona “MicroPython (generic)” y luego el puerto donde está conectado tu robot

Ahora, instala Pybricks en el robot. Abre una terminal en Thonny y escribe:

from pybricks import version
print(version)

Si responde con información, ¡ya está instalado!

Tip💡 Nota

Si no tienes el firmware instalado, visita https://docs.pybricks.com para las instrucciones completas de instalación [2].

Segundo: Escribir tu primer programa

Ahora vamos a escribir las instrucciones. Primero lo probaremos en la computadora, luego lo enviaremos al robot.

Un ejemplo simple en tu computadora

Abre Thonny y escribe esto:

print("Hola, bienvenidos al curso!")

Presiona F5 o el botón de Play. Verás el mensaje en la pantalla de abajo.

¡Felicidades! Acabas de escribir tu primer programa en Python.

Ahora, hacer lo mismo en el robot

Pero queremos que el robot lo haga, no la computadora. El código es similar, pero usamos funciones especiales del robot [2].

Copia esto en Thonny:

from pybricks.hubs import PrimeHub
from pybricks.parameters import Color
from pybricks.tools import wait

# Preparar el robot
hub = PrimeHub()

# Hacer que el robot diga "Hola"
hub.display.text("HOLA")
hub.light.on(Color.GREEN)
hub.speaker.beep()
wait(2000)

Tercero: Enviar el código al robot

Para que el robot ejecute nuestro programa:

  1. Conecta el robot por USB
  2. En Thonny, ve a Run → Select interpreter
  3. Elige el puerto donde está tu robot (parecerá algo como “COM3” en Windows o “/dev/tty.usb” en Mac)
  4. Presiona el botón de Play

¡El robot mostrará “HOLA”, will encenderse la luz verde y hará un sonido!

Misión de esta unidad

Ahora intenta esto:

  1. Cambia el mensaje de “HOLA” a tu nombre (máximo 5 letras)
  2. Cambia el color de la luz (prueba BLUE, RED, YELLOW)
  3. Intenta que el robot haga dos cosas seguidas

Cuando logres que el robot responda a tus instrucciones, habrás completado esta unidad.

En la siguiente unidad: cómo mostrar diferentes dibujos y mensajes en la pantalla del robot.
