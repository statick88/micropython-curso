Unidad 2: ¿Qué puede mostrar el robot?

⏱️ Tiempo estimado: 45 minutos | 📊 Nivel: Principiante

Antes de empezar

¿Has visto esos letreros de texto que cambian en las tiendas? O las pantallas de los semáforos que muestran diferentes símbolos. Bueno, el robot LEGO tiene algo parecido: una pequeña pantalla donde puede mostrar dibujos, letras, y números.

Imagina que tienes un cuaderno de puntitos (como el de matemática). Si coloreas algunos puntitos, puedes formar figuras. La pantalla del robot funciona igual: tiene 25 “puntitos” que puedes prender o apagar [1][2].

La pantalla del robot: un cuaderno de 5x5

Situación: Estás en un cine y tienes asignado el asiento F-3. Fila F, butaca 3. El robot tiene algo similar: “asientos” organizados en 5 filas y 5 columnas.

La pantalla tiene 25 "asientos" (puntitos LEDs):

    0   1   2   3   4  ← son las columnas
  ┌───┬───┬───┬───┐
0 │ • │ • │ • │ • │ • │ ← primera fila (arriba)
  ├───┼───┼───┼───┤
1 │ • │ • │ • │ • │ • │
  ├───┼───┼───┼───┤
2 │ • │ • │ • │ • │ • │ ← fila del medio
  ├───┼───┼───┼───┤
3 │ • │ • │ • │ • │ • │
  ├───┼───┼───┼───┤
4 │ • │ • │ • │ • │ • │ ← última fila (abajo)
  └───┴───┴───┴───┘

Cada “puntito” (LED) puede estar: - Apagado (0 = nada de luz) - Encendido (100 = brillo máximo)

Dibujos ya listos para usar

¿Recuerdas cuando en el celular tenías los emojis listos para usar? No tenías que dibujarlos, solo elegías el que querías.

El robot tiene algo similar—dibujos listos que puedes usar [2]:

# Primero, traer las herramientas
from pybricks.hubs import PrimeHub
from pybricks.tools import wait

hub = PrimeHub()

# Ahora puedes mostrar dibujos:
hub.display.icon("HAPPY")    # Carita feliz
hub.display.icon("SAD")      # Carita triste  
hub.display.icon("ARROW_UP")   # Flecha arriba
hub.display.icon("ARROW_DOWN") # Flecha abajo
hub.display.icon("HEART")    # Corazón

Prueba esto en Thonny:

from pybricks.hubs import PrimeHub
from pybricks.tools import wait

hub = PrimeHub()

# Mostrar carita feliz por 2 segundos
hub.display.icon("HAPPY")
wait(2000)

# Ahora mostrar un corazón
hub.display.icon("HEART")
wait(2000)

Crear tus propios dibujos

Situación: Ahora quieres dibujar algo que NO está en los dibujos listos. Por ejemplo, tu inicial.

Esto es como cuando dibujas en papel quadriculado: decides qué cuadros pintar y cuáles dejar en blanco.

Ejemplo: Hacer una cara sonriente

# Cada número es el brillo de un LED (0=apagado, 100=max)

# Así se ve una carita:
# ■ □ ■ □ ■  ← primera fila
# ■ □ ■ □ ■  ← segunda fila  
# □ □ ■ □ □  ← tercera fila (nariz)
# ■ □ □ □ ■  ← cuarta fila (boca)
# □ ■ ■ ■ □  ← quinta fila

# En código, se escribe así:
mi_dibujo = [
    [100, 0, 100, 0, 100],
    [100, 0, 100, 0, 100],
    [0, 0, 100, 0, 0],
    [100, 0, 0, 0, 100],
    [0, 100, 100, 100, 0],
]

# Enviar el dibujo al robot
hub.display.icon(mi_dibujo)
wait(2000)

¿Puedes dibujar tu inicial?

Intenta hacer una “A” o tu letra favorita. Solo necesitas decidir qué puntitos se prenden y cuáles se apaganal.

La luz del centro

Además de la pantalla, el robot tiene una luz en el centro—como un pequeño reflector [2].

# Encender la luz en diferentes colores
hub.light.on(5)   # Verde
hub.light.on(10)  # Naranja/amarillo
hub.light.on(0)   # Rojo

# O apagar la luz
hub.light.off()

¿Para qué sirve? Puedes usar la luz para indicar estados: - Verde = “todo bien” - Rojo = “algo mal” - Naranja = “espera”

Es como las luces de un semáforo o las luces de alerta de un auto.

Pero… ¿qué pasa si quiero mostrar algo, luego otra cosa, y otra más?

Situación: Estás viendo una animación en tu celular. Las imágenes van cambiando rápido, tan rápido que parece que se mueve.

¡El robot también puede hacer eso! [2]

# Primer dibujo
dibujo1 = [0, 0, 0, 0, 0]
dibujo2 = [100, 0, 0, 0, 0]
dibujo3 = [100, 100, 0, 0, 0]

# Mostrar uno tras otro (como una animación)
hub.display.icon(dibujo1)
wait(200)
hub.display.icon(dibujo2)
wait(200)
hub.display.icon(dibujo3)
wait(200)

Eso es una animación—mostrar varios dibujos seguidos muy rápido. Como las caricaturas que ves en televisión.
