Sensores Adicionales: Color, Fuerza y Distancia

⏱️ Tiempo estimado: 60 minutos | 📊 Nivel: Avanzado

Antes de empezar

Ya conoces el sensor ultrasónico que mide distancia usando sonido. Pero hay muchos más sensores que puedes usar con tu robot LEGO.

En esta unidad vas a aprender sobre sensores muy útiles disponibles en SPIKE Essential:

  1. Sensor de Color — Detecta colores y mide reflexión de luz
  2. Sensor de Fuerza — Mide presión aplicada
  3. Sensor de Distancia — Medición precisa en cm/pulgadas

1. Sensor de Color (ColorSensor)

Situación: Cuando vas a un semáforo, sabes si es rojo, amarillo o verde por el color. El robot también puede aprender a identificar colores.

Métodos disponibles

El ColorSensor tiene varios métodos para obtener diferentes tipos de información:

Método Retorna Descripción
get_color() str o None Color detectado
get_reflected_light() int (0-100) Porcentaje de luz reflejada
get_ambient_light() int (0-100) Intensidad de luz ambiental
get_red() int (0-1024) Intensidad del color rojo
get_green() int (0-1024) Intensidad del color verde
get_blue() int (0-1024) Intensidad del color azul
get_rgb_intensity() int (0-1024) Intensidad RGB combinada

Colores que puede detectar

El método get_color() retorna estos valores:

Valor Retornado Color Ejemplo de uso
'black' Negro Línea negra en el piso
'white' Blanco Línea blanca, piso claro
'red' Rojo Color primario
'green' Verde Color primario
'blue' Azul Color primario
'yellow' Amarillo Color primario
'cyan' Cian Color secundario
'violet' Violeta Color secundario
None Ninguno Sin color detectado

Código ejemplo básico

from spike import ColorSensor
from spike.control import wait

# Conectar el sensor de color al puerto C
sensor_color = ColorSensor('C')

while True:
    # Obtener el color detectado
    color = sensor_color.get_color()
    
    # Obtener reflexión de luz (0-100%)
    reflexion = sensor_color.get_reflected_light()
    
    print(f"Color: {color} | Reflexión: {reflexion}%")
    wait(100)

Tabla de reflexión de luz

La reflexión de luz es muy útil para seguir líneas o clasificar superficies:

Reflexión (%) Tipo de superficie Aplicación práctica
0-10 Muy oscuro Línea negra gruesa
10-20 Oscuro Línea negra delgada
20-35 Medio-oscuro Color azul/verde oscuro
35-50 Medio Colores intermedios
50-70 Claro Colores claros (amarillo, rojo claro)
70-85 Muy claro Blanco, piso claro
85-100 Máximo Espejo o superficie reflectiva

Ejemplo: Obtener valores RGB

from spike import ColorSensor
from spike.control import wait

sensor = ColorSensor('C')

while True:
    # Obtener componentes RGB individuales
    rojo = sensor.get_red()
    verde = sensor.get_green()
    azul = sensor.get_blue()
    intensidad = sensor.get_rgb_intensity()
    
    print(f"R: {rojo} | G: {verde} | B: {azul} | Total: {intensidad}")
    wait(200)

Control de luces LED del sensor

El sensor de color tiene LEDs integrados que puedes controlar:

from spike import ColorSensor
from spike.control import wait

sensor = ColorSensor('C')

# Encender los 3 LEDs individuales (0-100)
sensor.light_up(100, 0, 100)  # Rojo y azul (morado)
wait(1000)

# Encender todos los LEDs al mismo brillo
sensor.light_up_all(50)  # Brillo 50%
wait(1000)

# Apagar LEDs
sensor.light_up(0, 0, 0)

Proyecto: Seguidor de línea

from spike import ColorSensor
from spike import MotorPair
from spike.control import wait

# Configuración
sensor_linea = ColorSensor('C')
robot = MotorPair('A', 'B')

print("Iniciando seguidor de línea...")

# El robot sigue una línea negra
while True:
    reflexion = sensor_linea.get_reflected_light()
    
    if reflexion < 30:
        # Línea negra (oscuro) - girar izquierda
        robot.start_tank(30, 50)  # izquierdo más lento
    else:
        # Piso blanco (claro) - girar derecha
        robot.start_tank(50, 30)  # derecho más lento
    
    wait(10)

📝 Ejercicio para el Aula: Clasificador de Colores

Nivel: Básico | Tiempo: 15 minutos

Objetivo

El robot detecta diferentes colores y muestra un icono diferente en la pantalla LED para cada uno.

Solución

from spike import ColorSensor, PrimeHub
from spike.control import wait

hub = PrimeHub()
sensor = ColorSensor('C')

# Mapa de colores a iconos
iconos = {
    'red': 'HEART',
    'green': 'YES',
    'blue': 'NOTE',
    'yellow': 'DIAMOND',
    'black': 'SAD',
    'white': 'HAPPY'
}

while True:
    color = sensor.get_color()
    
    if color in iconos:
        hub.light_matrix.show_image(iconos[color])
        print(f"Detectado: {color}")
    else:
        hub.light_matrix.show_image('CONFUSED')
        print("Color no reconocido")
    
    wait(500)

Evaluación

Criterio Puntos Descripción
Detecta al menos 4 colores 25 Funciona con rojo, verde, azul, amarillo
Muestra icono correcto 25 Cada color tiene su icono
Código organizado 25 Usa diccionario para mapear
Funciona sin errores 25 Sinbugs en ejecución

2. Sensor de Distancia (DistanceSensor)

Situación: El sensor ultrasónico usa ondas de sonido para medir la distancia a objetos. Es como los murciélagos que usan el sonido para “ver” en la oscuridad.

Métodos disponibles

Método Parámetros Retorna Descripción
get_distance_cm(short_range) short_range (bool, opcional) int (0-200) Distancia en centímetros
get_distance_inches(short_range) short_range (bool, opcional) int (0-79) Distancia en pulgadas
get_distance_percentage(short_range) short_range (bool, opcional) int (0-100) Distancia como porcentaje

Parámetro short_range

Modo Rango Uso recomendado
False (default) 0-200 cm Mediciones normales
True 0-20 cm Objetos muy cercanos

Tabla de distancias

Distancia (cm) Descripción Acción del robot
0-5 Muy cerca Detenerse inmediatamente
5-15 Cerca Reducir velocidad
15-50 Medio Moverse con cuidado
50-100 Lejos Velocidad normal
100-200 Muy lejos Velocidad máxima

Código ejemplo

from spike import DistanceSensor
from spike.control import wait

# Inicializar el sensor de distancia en puerto B
sensor = DistanceSensor('B')

while True:
    # Obtener distancia en centímetros
    distancia = sensor.get_distance_cm()
    
    # Obtener como porcentaje (100% = muy lejos)
    porcentaje = sensor.get_distance_percentage()
    
    # Obtener en pulgadas
    pulgadas = sensor.get_distance_inches()
    
    print(f"Distancia: {distancia} cm ({pulgadas} in) - {porcentaje}%")
    wait(500)

Usar short_range (medición cercana)

from spike import DistanceSensor
from spike.control import wait

sensor = DistanceSensor('B')

while True:
    # Modo rango corto (0-20 cm)
    distancia_corta = sensor.get_distance_cm(short_range=True)
    
    if distancia_corta < 5:
        print("¡MUY CERCA!")
    else:
        print(f"Distancia: {distancia_corta} cm")
    
    wait(200)

Control de LEDs del sensor

from spike import DistanceSensor
from spike.control import wait

sensor = DistanceSensor('B')

# Encender los 4 LEDs individuales
# (right_top, left_top, right_bottom, left_bottom)
sensor.light_up(100, 0, 100, 0)  # Patrón alternativo
wait(1000)

# Encender todos
sensor.light_up_all(100)
wait(1000)

# Apagar
sensor.light_up_all(0)

Proyecto: Robot evita obstáculos

from spike import DistanceSensor, PrimeHub
from spike import MotorPair
from spike.control import wait

hub = PrimeHub()
sensor = DistanceSensor('B')
robot = MotorPair('A', 'B')

print("Robot evitando obstáculos...")

while True:
    distancia = sensor.get_distance_cm()
    
    if distancia < 15:
        # Obstáculo detectado - girar
        robot.stop()
        hub.light_matrix.show_image('SAD')
        hub.speaker.beep(60, 0.2)  # Beep de alerta
        
        # Girar a la derecha
        robot.move_tank(90, 'degrees', left_speed=50, right_speed=-50)
        wait(500)
    else:
        # No hay obstáculo - avanzar
        hub.light_matrix.show_image('HAPPY')
        robot.start(30)
    
    wait(100)

📝 Ejercicio para el Aula: Medidor de Distancia

Nivel: Intermedio | Tiempo: 20 minutos

Objetivo

El robot funciona como un medidor de distancia. Muestra diferentes iconos según qué tan lejos está el objeto.

Tabla de referencia

Distancia Icono Color LED
< 10 cm ANGRY Rojo
10-30 cm SAD Naranja
30-60 cm CONFUSED Amarillo
60-100 cm HAPPY Verde
> 100 cm YES Azul

Solución

from spike import DistanceSensor, PrimeHub
from spike import MotorPair
from spike.control import wait

hub = PrimeHub()
sensor = DistanceSensor('B')
robot = MotorPair('A', 'B')

def mostrar_distancia(distancia):
    """Muestra icono según la distancia"""
    if distancia < 10:
        hub.light_matrix.show_image('ANGRY')
    elif distancia < 30:
        hub.light_matrix.show_image('SAD')
    elif distancia < 60:
        hub.light_matrix.show_image('CONFUSED')
    elif distancia < 100:
        hub.light_matrix.show_image('HAPPY')
    else:
        hub.light_matrix.show_image('YES')
    
    print(f"Distancia: {distancia} cm")

# Loop principal
while True:
    distancia = sensor.get_distance_cm()
    mostrar_distancia(distancia)
    wait(300)

3. Sensor de Fuerza (ForceSensor)

Situación: Cuando aplicas presión en un botón de un cajero automático, sientes que “hace clic”. El sensor de fuerza puede medir exactamente cuánta presión estás aplicando.

Nota: El sensor de fuerza de SPIKE Essential puede no estar disponible en todos los kits. Verifica tu kit antes de usar.

Métodos disponibles

Método Retorna Descripción
is_pressed() bool True si hay presión aplicada
get_force() int Fuerza en Newton’s (aproximado)
wait_until_pressed() None Espera hasta que se presione
wait_until_released() None Espera hasta que se suelte

Tabla de fuerza

Estado Valor Aplicación
Sin presión 0 Robot en espera
Toque ligero 1-3 N Botón suave
Presión normal 3-7 N Presión típica
Presión fuerte 7-10 N Botón de emergencia
Presión máxima >10 N Máximo soportado

Código ejemplo

from spike import ForceSensor
from spike.control import wait

# Conectar el sensor de fuerza al puerto C
sensor_fuerza = ForceSensor('C')

while True:
    # Verificar si está presionado
    if sensor_fuerza.is_pressed():
        print("¡Presión detectada!")
    else:
        print("Sin presión")
    
    # Obtener valor de fuerza (si disponible)
    try:
        fuerza = sensor_fuerza.get_force()
        print(f"Fuerza: {fuerza}")
    except:
        pass  # Algunos modelos no soportan get_force()
    
    wait(100)

Ejemplo: Esperar hasta presionar

from spike import ForceSensor, PrimeHub
from spike.control import wait

hub = PrimeHub()
sensor = ForceSensor('C')

print("Presiona el sensor de fuerza...")

# Esperar hasta que se presione
sensor.wait_until_pressed()

# Cuando llegue aquí, ya se presionó
hub.light_matrix.show_image('HEART')
hub.speaker.beep(60, 0.5)
print("¡Detectado!")

📝 Ejercicio para el Aula: Contador de Presiones

Nivel: Básico | Tiempo: 15 minutos

Objetivo

El robot cuenta cuántas veces se presiona el sensor de fuerza y muestra el contador en la pantalla LED.

Solución

from spike import ForceSensor, PrimeHub
from spike.control import wait

hub = PrimeHub()
sensor = ForceSensor('C')

contador = 0
presionado_antes = False

print("Iniciando contador de presiones...")

while True:
    # Verificar estado actual
    esta_presionado = sensor.is_pressed()
    
    # Detectar flanco (cuando pasa de no presionado a presionado)
    if esta_presionado and not presionado_antes:
        contador += 1
        hub.light_matrix.show_image('HEART')
        print(f"Presión #{contador}")
        wait(200)  # Evitar doble conteo
    elif not esta_presionado:
        hub.light_matrix.show_image('SQUARE')
    
    presionado_antes = esta_presionado
    
    # Mostrar número (si es pequeño)
    if contador < 10:
        hub.light_matrix.write(str(contador))
    
    wait(50)

# Nota: Este código puede variar según la versión de SPIKE

4. Ejercicios de Práctica Integrados

Ejercicio 1: Robot que evita colores

Objetivo: El robot avanza pero se detiene cuando detecta color rojo.

from spike import ColorSensor
from spike import MotorPair
from spike.control import wait

sensor = ColorSensor('C')
robot = MotorPair('A', 'B')

while True:
    color = sensor.get_color()
    
    if color == 'red':
        # ¡Color rojo! Detenerse
        robot.stop()
        print("¡PELIGRO!")
        wait(1000)
    else:
        # Seguir adelante
        robot.start(30)
    
    wait(100)

Ejercicio 2: Contador de colores

Objetivo: El robot cuenta cuántos objetos de cada color detecta.

from spike import ColorSensor
from spike.control import wait

sensor = ColorSensor('C')

# Contadores para cada color
contadores = {
    'red': 0,
    'green': 0,
    'blue': 0,
    'yellow': 0,
    'other': 0
}

color_anterior = 'none'

while True:
    color = sensor.get_color()
    
    # Solo contar cuando cambia de color
    if color != color_anterior and color is not None:
        if color in contadores:
            contadores[color] += 1
        else:
            contadores['other'] += 1
        
        print(f"¡Nuevo {color}! Total: {contadores}")
        color_anterior = color
    
    wait(200)

Ejercicio 3: Robot detector de superficies

Objetivo: El robot identifica diferentes superficies según su reflexión.

from spike import ColorSensor
from spike.control import wait

sensor = ColorSensor('C')

def clasificar_superficie(reflexion):
    """Clasifica la superficie según el valor de reflexión"""
    if reflexion < 20:
        return "OSCURO (negro/piso oscuro)"
    elif reflexion < 45:
        return "MEDIO (azul/verde)"
    elif reflexion < 70:
        return "CLARO (amarillo/rojo claro)"
    else:
        return "MUY CLARO (blanco/piso claro)"

print("Detectando superficies...")

while True:
    reflexion = sensor.get_reflected_light()
    tipo = clasificar_superficie(reflexion)
    
    print(f"Reflexión: {reflexion}% -> {tipo}")
    wait(1000)
