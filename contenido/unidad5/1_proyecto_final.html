Fase 5: Despliegue Final (Proyecto)

El Protocolo Final: Arquitectura de Exploración

Has llegado a la fase de Maestría. Como Arquitecto Principal, tu misión es diseñar un sistema robótico integrado que demuestre autonomía, precisión y telemetría avanzada. Ya no estamos escribiendo scripts; estamos desplegando un Firmware de Misión.

1. El Desafío: Protocolo “Search & Report”

El robot debe ejecutar una patrulla autónoma en un área rectangular, detectando obstáculos y reportando el estado del entorno mediante el HUD y la telemetría sonora.

Requisitos de Ingeniería

  1. Core de Navegación: Uso de DriveBase con use_gyro(True).
  2. Concurrencia: Una tarea de movimiento y una tarea de supervisión de sensores.
  3. Modularidad: El código debe estar organizado en funciones claras.
  4. Resiliencia: Manejo de errores para evitar bloqueos del sistema.

2. Plantilla de Arquitectura Maestra

Utiliza esta estructura como base para tu despliegue. Observa cómo integramos todo lo aprendido en las fases 1 a 4.

from pybricks.hubs import PrimeHub # Cerebro del sistema
from pybricks.pupdevices import Motor, ColorSensor, UltrasonicSensor # Actuadores y Sensores
from pybricks.parameters import Port, Color, Icon, Direction # Configuración técnica
from pybricks.robotics import DriveBase # Lógica cinemática
from pybricks.tools import wait, run_task # Orquestador de tareas

# --- CONFIGURACIÓN DE SUBSISTEMAS --- #
hub = PrimeHub()
# Inicializar base de tracción con dimensiones exactas
robot = DriveBase(Motor(Port.A, Direction.COUNTERCLOCKWISE), Motor(Port.B), 56, 112)
sensor_dist = UltrasonicSensor(Port.D)
sensor_col = ColorSensor(Port.C)

1async def supervision_telemetria():
    """Monitoreo constante de variables críticas del entorno."""
    while True:
        dist = sensor_dist.distance() # Medir proximidad en mm
        if dist < 100:
            # Protocolo de emergencia ante obstáculo detectado
2            hub.light.on(Color.RED)
            hub.display.icon(Icon.STOP) # Alerta visual en el HUD
        else:
            hub.light.on(Color.GREEN) # Estado nominal
        await wait(50) # Liberar CPU para otros procesos

3async def protocolo_mision():
    """Orquestador del flujo principal de navegación autónoma."""
    robot.use_gyro(True) # Activar sensor inercial para rumbo fijo
    print("LOG: Iniciando Patrulla de Ingeniería...")
    
    # Ciclo de patrulla rectangular (4 lados)
    for i in range(4):
        hub.display.number(i + 1) # Reportar progreso en HUD
4        await robot.straight(400)
        await robot.turn(90) # Rotación asíncrona de 90 grados
    
    print("LOG: Misión Finalizada con Éxito.")
    hub.display.text("DONE") # Feedback final

if __name__ == "__main__":
    try:
        # Iniciar el ecosistema completo de procesos
5        run_task(protocolo_mision(), supervision_telemetria())
    except Exception as e:
        # Captura de fallos para diagnóstico de campo
6        print(f"FALLO CRÍTICO EN EL SISTEMA: {e}")
