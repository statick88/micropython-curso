¿Qué podrías crear?

Situación: Piensa en algo que un robot podría hacer. No tiene que ser complicado—puede ser algo simple.

Aquí hay algunas ideas:

Proyecto Qué hace Para qué sirve
Explorador El robot camina y evita chocar Como un robot de limpieza
Vigilante El robot hace un camino y observa Como un guardia que ronda
Contador El robot avanza y cuenta pasos Para medir distancias
Respondedor El robot hace algo diferente según lo que detecte Como un juego

Ejemplo: Robot Explorador Completo

Aquí hay un ejemplo de algo que podrías hacer—un robot que explora un área y evita chocar [2][3][4]:

"""
PROYECTO: Robot Explorador
El robot camina en un rectángulo, detectando obstáculos.
"""

# 1. Herramientas
from pybricks.hubs import PrimeHub
from pybricks.pupdevices import Motor, UltrasonicSensor
from pybricks.parameters import Port
from pybricks.robotics import DriveBase
from pybricks.tools import wait, run_task

# 2. Preparar el robot
hub = PrimeHub()
motor_izq = Motor(Port.A)
motor_der = Motor(Port.B)
sensor = UltrasonicSensor(Port.D)
robot = DriveBase(motor_izq, motor_der, 56, 112)

# Variable para comunicar entre partes
hay_obstaculo = False

# 3. Parte 1: Vigilar (como estar atento en clase)
async def vigilante():
    global hay_obstaculo
    
    while True:
        distancia = sensor.distance()
        
        if distancia < 150:  # menos de 15cm
            hay_obstaculo = True
            hub.light.on(0)  # luz roja = cuidado
        else:
            hay_obstaculo = False
            hub.light.off()
        
        wait(50)

# 4. Parte 2: Moverse (como caminar por el pasillo)
async def explorador():
    global hay_obstaculo
    
    robot.use_gyro(True)
    
    print("¡Explorador iniciando!")
    
    # Hacer 4 lados (como un rectángulo)
    for lado in range(4):
        print(f"Lado {lado + 1}")
        
        # Avanzar un poco varias veces
        for paso in range(4):
            if hay_obstaculo:
                robot.stop()
                await robot.turn(90)  # girar y buscar otro camino
                await wait(200)
            else:
                await robot.straight(100)  # avanzar 10cm
        
        # Girar en la esquina
        await robot.turn(90)
    
    robot.stop()
    print("¡Exploración terminada!")

# 5. Ejecutar ambas partes
run_task(explorador(), vigilante())

¿Qué hace este robot? 1. Camina en un rectángulo (4 lados) 2. Mientras camina, observa si hay algo adelante [4] 3. Si detecta algo, se detiene, gira, y sigue por otro lado

¿Qué debe tener tu proyecto?

Tu proyecto debe incluir:

Qué debe tener Para qué
Que se mueva El robot va de un lugar a otro [3]
Que detecte algo El robot usa sus “ojos” (sensor) [4]
Que tome decisiones Si detecta algo, hace algo diferente
Que esté organizado El código tiene partes claras
Que se entienda Explicas qué hace tu programa

Pasos para hacer tu proyecto

Paso 1: Decide qué quieres hacer - Piensa en un problema que el robot pueda resolver - No tiene que ser complicado—empieza simple

Paso 2: Escribe el código - Usa las herramientas que ya conoces [1][2] - Prueba por partes si es necesario

Paso 3: Prueba y ajusta - ¿Funciona como esperabas? - ¿Qué puedes mejorar?

Paso 4: Presenta tu proyecto - Explica qué hace tu robot - Muestra cómo funciona

💡 Consejos

  • Empieza simple: Primero haz que funcione algo básico, luego agrega más cosas
  • Prueba por partes: Si algo no funciona, revisa una parte a la vez
  • Los errores son parte de aprender: Si algo no funciona, está bien—así se aprende
  • Usa lo que ya sabes: No hay nada nuevo que necesitas—todo ya lo aprendiste
