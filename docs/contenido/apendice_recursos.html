1. ¿No tienes acceso a LEGO? Aquí hay alternativas

Entendemos que el hardware de LEGO puede ser costoso. Existen alternativas más económicas que funcionan con MicroPython [1]:

1.1 micro:bit V2 — La opción más económica

Precio: $20-25 USD | Python: MicroPython nativo

El BBC micro:bit es una微型computadora diseñada específicamente para educación. Tiene sensores integrados, LEDs y botones [2].

┌─────────────────────────────────────────┐
│           micro:bit V2                  │
│  ┌─────────────────────────────────┐    │
│  │ ● ● ● ● ●  (matriz 5x5 LEDs)   │    │
│  └─────────────────────────────────┘    │
│  [A]              [B]                   │
│  ┌─────────────────────────────────┐    │
│  │ 🎤 💧 🌡️ 📡 🔌 (sensores)       │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘

Características: - Micrófono integrado - Speaker integrado - Acelerómetro - Magnetómetro - Bluetooth - Radio

Código ejemplo:

from microbit import *

# Mostrar un corazón
display.show(Image.HEART)

# Botones
while True:
    if button_a.is_pressed():
        display.show("A")
    elif button_b.is_pressed():
        display.show("B")

Enlaces: - 📖 microbit.org — Sitio oficial [2] - 📖 microbit.org/python — Documentación Python [2]

1.2 Raspberry Pi Pico — Potencia a bajo costo

Precio: $4-10 USD | Python: MicroPython nativo

El Raspberry Pi Pico es un microcontrolador muy potente diseñado por Raspberry Pi Foundation [3].

┌─────────────────────────────────────────┐
│        Raspberry Pi Pico                │
│                                         │
│  ○ ○ ○ ○  ───  USB                     │
│  GND VBUS VSYS   (pins)                │
│  GP0  GP1  ...                         │
│                                         │
│  ────────────  Botón                  │
└─────────────────────────────────────────┘

Características: - Dual-core ARM Cortex-M0+ - 264 KB RAM - 2 MB Flash - 40 pines GPIO - ADC, PWM, SPI, I2C

Código ejemplo:

from machine import Pin, PWM
from time import sleep

# LED incorporado
led = Pin(25, Pin.OUT)

# Parpadear
while True:
    led.toggle()
    sleep(1)

# LED externo en PWM
led_externo = PWM(Pin(15))
led_externo.freq(1000)

for duty in range(0, 65536, 1000):
    led_externo.duty_u16(duty)
    sleep(0.1)

Enlaces: - 📖 raspberrypi.com/pico — Comprar [3] - 📖 raspberrypi.com/documentation/pico-sdk — Documentación [3]

1.3 Arduino Alvik — La alternativa moderna

Precio: $99 USD | Python: MicroPython nativo

El Arduino Alvik es un robot educativo lanzado en 2024, diseñado específicamente para aprender MicroPython y robótica [4].

┌─────────────────────────────────────────┐
│           Arduino Alvik                │
│      ┌────────────────────┐             │
│     ╱  ● ○ ●  (sensores)  ╲            │
│    ╱   ┌──┐   ┌──┐   ┌──┐    ╲          │
│   │    │▓▓│   │▓▓│   │▓▓│     │         │
│   │    └──┘   └──┘   └──┘     │         │
│   │     RUEDAS           │         │
│   │                      │         │
│   └──────────────────────┘         │
└─────────────────────────────────────────┘

Características: - Sensores de línea integrados - Sensor de distancia ToF - Acelerómetro - LED RGB - Batería integrada - Wheels incluidos

Código ejemplo:

from arduino import robot

# Mover el robot
robot.forward(10)  # 10 cm
robot.turn(90)     # 90 grados

# Sensores
distancia = robot.distance()
linea = robot.line_sensor()

# Mostrar en pantalla
robot.display.show("HOLA")

Enlaces: - 📖 arduino.cc/alvik — Sitio oficial [4]

2. Simuladores: Practicar sin hardware

Si no tienes el robot físico, puedes practicar con simuladores [1]:

2.1 Thonny + Python en PC (sin robot)

Puedes ejecutar código Python en tu computadora sin el robot:

# Este código funciona en la computadora
# (aunque no controlará un robot físico)

# Simular el robot
class SimulatedHub:
    def __init__(self):
        self.luz_estado = False
        self.texto = ""
    
    def light_on(self, color):
        print(f"🔵 LEDendido (color {color})")
        self.luz_estado = True
    
    def display_text(self, texto):
        print(f"📺 Pantalla: {texto}")
        self.texto = texto

# Usar como si fuera el robot real
hub = SimulatedHub()
hub.light_on(5)
hub.display_text("PRACTICA")

2.2 OpenBot (Smartphone + DIY)

OpenBot usa un teléfono Android como cerebro del robot—mucho más económico [5].

Precio: ~$50 (materiales) + smartphone (ya tienes)

Enlaces: - 📖 github.com/isl-org/OpenBot — Repositorio oficial [5]
