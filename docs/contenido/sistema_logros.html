Método 2: En el Código

# Sistema simple de tracking en MicroPython
from spike import PrimeHub

hub = PrimeHub()

# Tu progreso actual
XP_ACTUAL = 0
NIVEL = 1
INSIGNIAS = []

def ganar_xp(puntos, insignia):
    global XP_ACTUAL, NIVEL
    XP_ACTUAL += puntos
    INSIGNIAS.append(insignia)
    
    # Verificar si sube de nivel
    niveles = [0, 100, 250, 500, 750, 1000, 1500]
    for i, xp_nivel in enumerate(niveles):
        if XP_ACTUAL >= xp_nivel:
            NIVEL = i + 1
    
    # Mostrar progreso
    print(f"¡{insignia} +{puntos} XP!")
    print(f"Nivel {NIVEL} - {XP_ACTUAL} XP total")

# Ejemplo de uso
ganar_xp(25, "🐍 Primer Código")

🎮 Reglas del Juego

  1. XP es acumulativo - No se pierde al fallar un quiz
  2. Reintentos permitidos - Puedes volver a intentar quizzes
  3. Bonus por excelencia - Score 100% da insignia extra
  4. Creatividad premiada - Proyectos originales ganan XP bonus
  5. Colaboración valiosa - Ayudar a otros suma puntos

📊 Cálculo de Nota Final

La gamificación complementa la calificación tradicional:

Componente Peso Gamificación
Quizzes 30% 📝 Quiz Dominado × 5
Labs 30% 🧪 Completar todos los labs
Proyecto Final 30% 🏆 Proyecto aprobado
Participación 10% 💬 Bonus por insignias

Fórmula:

Nota Final = (Quizzes × 0.3) + (Labs × 0.3) + (Proyecto × 0.3) + (Participación × 0.1)

¡Recuerda: El objetivo es APRENDER, no solo acumular puntos! Los puntos son una herramienta, no el fin. 🎓
