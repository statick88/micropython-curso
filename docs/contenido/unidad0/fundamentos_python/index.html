Módulo de Fundamentos de Python para Robótica Educativa

Este módulo proporciona los fundamentos de programación en Python necesarios para trabajar con robótica educativa utilizando LEGO Spike Prime. Está diseñado siguiendo la metodología de enseñanza de Diego Saavedra, donde los conceptos se introducen primero con analogías del mundo real antes de aplicarlos a la programación.

Objetivos de Aprendizaje

Al completar este módulo, los estudiantes serán capaces de:

  1. Comprender y utilizar variables para almacenar información que cambia durante la ejecución del programa
  2. Aplicar estructuras de control (condicionales y bucles) para tomar decisiones y repetir acciones
  3. Crear funciones para organizar código reutilizable
  4. Manejar errores de manera elegante cuando las cosas no salen como esperamos
  5. Utilizar estructuras de datos (listas y diccionarios) para organizar información compleja

Estructura del Módulo

El módulo está organizado en 6 lecciones secuenciales que construyen conocimientos de manera acumulativa:

Lecciones

Lección Tema Descripción
1 ¿Por qué aprender Python primero? Justificación del enfoque de fundamentos antes de hardware
2 Variables y Tipos de Datos Cómo guardar y organizar información
3 Estructuras de Control Condicionales y bucles para decisiones y repeticiones
4 Funciones Crear “recetas” reutilizables de código
5 Manejo de Errores Qué hacer cuando las cosas fallan
6 Estructuras de Datos Listas y diccionarios para información compleja
7 Resumen y Transición Preparación para trabajar con hardware real

Metodología

Este módulo sigue los principios pedagógicos establecidos:

  • Analogías Primero: Cada concepto se introduce con ejemplos de la vida cotidiana
  • Método Socrático: Preguntas guiantes para fomentar el descubrimiento
  • Código como Comunicación: Nombres descriptivos y comentarios significativos
  • Aprendizaje Basado en Desafíos: 70% práctica, 30% teoría
  • Enfoque Práctico sin Hardware: Ejercicios que anticipan la aplicación en robótica

Requisitos Previos

No se requiere experiencia previa en programación. Este módulo está diseñado para absolute beginners que nunca han programado antes.

Requisitos para Completar

Para completar este módulo necesitas:

  • Acceso a un intérprete de Python (Thonny, terminal de Python, o https://python.org/shell/)
  • Navegador web para acceder a recursos externos
  • Ganas de aprender y experimentar

Tiempo Estimado

  • Lectura y comprensión: 4-6 horas
  • Ejercicios prácticos: 6-8 horas
  • Total: 10-14 horas distribuidas en 1-2 semanas

Evaluación

Al final de cada lección hay ejercicios de verificación de comprensión. También hay mini-desafíos que integran los conceptos aprendidos.

Próximo Módulo

Una vez completado este módulo, continúa con la Unidad 1: El Hub LEGO - Desde Cero donde começarás a trabajar con el hardware real.
