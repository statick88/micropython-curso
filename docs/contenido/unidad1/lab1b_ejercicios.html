Ejercicio 1.1: Secuencia de LEDs

Descripción

Crea una secuencia de LEDs que simule un “carrusel de luces” recorriendo la matriz de 5x5.

Instrucciones

  1. Enciende un LED a la vez
  2. Recorre todas las posiciones de izquierda a derecha, fila por fila
  3. Usa un retraso de 100ms entre cada paso
  4. Repite la secuencia 3 veces

Código Base

from spike import PrimeHub
from spike.control import wait

hub = PrimeHub()

# Tu código aquí
# Pista: usa una matriz 5x5 con valores 0 y 100

Solución

from spike import PrimeHub
from spike.control import wait

hub = PrimeHub()

# Secuencia de carrusel de luces
for repeticion in range(3):
    for fila in range(5):
        for columna in range(5):
            # Crear matriz con un solo LED encendido
            matriz = [[0, 0, 0, 0, 0] for _ in range(5)]
            matriz[fila][columna] = 100
            hub.light_matrix.show(matriz)
            wait(100)
    
    # Pausa entre repeticiones
    hub.light_matrix.off()
    wait(500)

Conceptos Clave

Concepto Descripción
Matriz 5x5 Lista de listas con valores 0-100
Bucles anidados Recorrer filas y columnas
wait() Pausa no bloqueante en milisegundos

Ejercicio 1.2: Patrón de Respiración

Descripción

Implementa un efecto de “respiración” donde la matriz completa aumenta y disminuye su brillo suavemente.

Instrucciones

  1. Enciende todos los LEDs al mínimo (brillo 10)
  2. Aumenta gradualmente hasta el máximo (brillo 100)
  3. Disminuye gradualmente hasta el mínimo
  4. Repite el ciclo 5 veces

Código Base

from spike import PrimeHub
from spike.control import wait

hub = PrimeHub()

# Tu código aquí
# Pista: usa range() con paso para controlar el brillo

Solución

from spike import PrimeHub
from spike.control import wait

hub = PrimeHub()

for ciclo in range(5):
    # Fase de inhalación: aumentar brillo
    for brillo in range(10, 101, 5):
        matriz = [[brillo for _ in range(5)] for _ in range(5)]
        hub.light_matrix.show(matriz)
        wait(30)
    
    # Fase de exhalación: disminuir brillo
    for brillo in range(100, 9, -5):
        matriz = [[brillo for _ in range(5)] for _ in range(5)]
        hub.light_matrix.show(matriz)
        wait(30)

hub.light_matrix.off()

Ejercicio 1.3: Interacción con Botones

Descripción

Crea un programa interactivo donde los botones del Hub controlan diferentes comportamientos.

Instrucciones

  1. Botón LEFT: Muestra una flecha apuntando izquierda
  2. Botón RIGHT: Muestra una flecha apuntando derecha
  3. Botón CENTER: Muestra un corazón
  4. Botón LEFT + RIGHT: Sale del programa

Código Base

from spike import PrimeHub

hub = PrimeHub()

# Define los iconos como matrices
# Pista: usa hub.light_matrix.show() con matrices 5x5

Solución

from spike import PrimeHub

hub = PrimeHub()

# Iconos personalizados (matrices 5x5)
FLECHA_IZQ = [
    [0, 0, 100, 0, 0],
    [0, 100, 100, 0, 0],
    [100, 100, 100, 100, 100],
    [0, 100, 100, 0, 0],
    [0, 0, 100, 0, 0]
]

FLECHA_DER = [
    [0, 0, 100, 0, 0],
    [0, 0, 100, 100, 0],
    [100, 100, 100, 100, 100],
    [0, 0, 100, 100, 0],
    [0, 0, 100, 0, 0]
]

CORAZON = [
    [100, 0, 100, 0, 100],
    [100, 100, 100, 100, 100],
    [100, 100, 100, 100, 100],
    [0, 100, 100, 100, 0],
    [0, 0, 100, 0, 0]
]

while True:
    # Salir con LEFT + RIGHT
    if hub.left_button.is_pressed() and hub.right_button.is_pressed():
        hub.light_matrix.off()
        break
    
    # Mostrar iconos según botón
    if hub.left_button.is_pressed():
        hub.light_matrix.show(FLECHA_IZQ)
    elif hub.right_button.is_pressed():
        hub.light_matrix.show(FLECHA_DER)
    elif hub.center_button.is_pressed():
        hub.light_matrix.show(CORAZON)
    
    wait(50)

Ejercicio 1.4: Contador Binario

Descripción

Muestra un contador binario del 0 al 31 usando los primeros 5 LEDs de la primera fila.

Instrucciones

  1. Cuenta del 0 al 31
  2. Representa cada número en binario usando la primera fila
  3. Cada LED representa un bit (apagado=0,endido=100)
  4. Retraso de 500ms entre cada número

Pista

El número 5 en binario es 00101. Los bits van de derecha a izquierda.

Solución

from spike import PrimeHub
from spike.control import wait

hub = PrimeHub()

for numero in range(32):
    # Convertir número a representación binaria
    matriz = [[0, 0, 0, 0, 0] for _ in range(5)]
    
    # Llenar la primera fila con bits
    for bit in range(5):
        if numero & (1 << bit):  # Operación AND bitwise
            matriz[0][bit] = 100
    
    hub.light_matrix.show(matriz)
    wait(500)

hub.light_matrix.off()

Ejercicio 1.5: Sonido Melódico

Descripción

Programa una melodía simple usando el speaker del Hub.

Instrucciones

  1. Toca 4 notas ascendentes (DO, RE, MI, FA)
  2. Toca 4 notas descendentes (FA, MI, RE, DO)
  3. Toca un beep final

Código Base

from spike import PrimeHub

hub = PrimeHub()

# Usa hub.speaker.start_beep()
# Pista: frecuencia 262 = DO4, 294 = RE4, 330 = MI4, 349 = FA4

Solución

from spike import PrimeHub
from spike.control import wait

hub = PrimeHub()

# Escala ascendente
hub.speaker.start_beep(262)  # DO4
wait(250)
hub.speaker.start_beep(294)  # RE4
wait(250)
hub.speaker.start_beep(330)  # MI4
wait(250)
hub.speaker.start_beep(349)  # FA4
wait(500)

# Escala descendente
hub.speaker.start_beep(349)  # FA4
wait(250)
hub.speaker.start_beep(330)  # MI4
wait(250)
hub.speaker.start_beep(294)  # RE4
wait(250)
hub.speaker.start_beep(262)  # DO4
wait(500)

# Beep final
hub.speaker.beep()

Ejercicio 1.6: Menú Interactivo

Descripción

Combina todo lo aprendido para crear un menú interactivo con el Hub.

Requisitos

  1. Pantalla inicial con icono de “menú”
  2. CENTER selecciona opción
  3. LEFT/RIGHT navega entre opciones
  4. Opciones:
    • Opción 1: Secuencia de LEDs (Ejercicio 1.1)
    • Opción 2: Respiración (Ejercicio 1.2)
    • Opción 3: Melodía (Ejercicio 1.5)

Solución

from spike import PrimeHub
from spike.control import wait

hub = PrimeHub()

# Imágenes predefinidas de SPIKE Essential
IMAGENES = ['ARROW_LEFT', 'ARROW_RIGHT', 'YES']
NOMBRES = ["LEDs", "Respiración", "Melodía"]

opcion_actual = 0

while True:
    # Mostrar opción actual
    hub.light_matrix.show_image(IMAGENES[opcion_actual])
    
    # Navegación
    if hub.left_button.is_pressed():
        opcion_actual = (opcion_actual - 1) % 3
        wait(200)
    elif hub.right_button.is_pressed():
        opcion_actual = (opcion_actual + 1) % 3
        wait(200)
    elif hub.center_button.is_pressed():
        # Ejecutar opción
        if opcion_actual == 0:
            # Secuencia LEDs
            for _ in range(3):
                for f in range(5):
                    for c in range(5):
                        m = [[0]*5 for _ in range(5)]
                        m[f][c] = 100
                        hub.light_matrix.show(m)
                        wait(100)
        elif opcion_actual == 1:
            # Respiración
            for _ in range(3):
                for b in range(10, 101, 5):
                    m = [[b]*5 for _ in range(5)]
                    hub.light_matrix.show(m)
                    wait(20)
                for b in range(100, 9, -5):
                    m = [[b]*5 for _ in range(5)]
                    hub.light_matrix.show(m)
                    wait(20)
        elif opcion_actual == 2:
            # Melodía
            hub.speaker.start_beep(262)
            wait(250)
            hub.speaker.start_beep(294)
            wait(250)
            hub.speaker.start_beep(330)
            wait(250)
            hub.speaker.start_beep(349)
    
    wait(50)
