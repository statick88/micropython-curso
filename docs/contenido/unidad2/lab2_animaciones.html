Laboratorio 2.1: Animaciones en Matriz 5x5

⚡ Nivel: Intermedio | ⏱️ Tiempo: 35 minutos

Objetivo del Laboratorio

Dominar el control pixel a pixel de la matriz de 5x5 LED del Hub para crear animaciones personalizadas que comuniquen estados del sistema en tiempo real.

Materiales Requeridos

Componente Especificación
Hub LEGO SPIKE Prime
Cable USB LEGO
PC Con Pybricks IDE

⚡ Procedimiento Paso a Paso

Paso 1: Entendiendo la Matriz (5 minutos)

La matriz 5x5 tiene 25 LEDs organizados en coordenadas (x, y):

Coordenadas de la matriz 5x5:
┌───┬───┬───┬───┬───┐
│0,0│1,0│2,0│3,0│4,0│  ← Fila 0 (superior)
├───┼───┼───┼───┼───┤
│0,1│1,1│2,1│3,1│4,1│
├───┼───┼───┼───┼───┤
│0,2│1,2│2,2│3,2│4,2│  ← Centro (2,2)
├───┼───┼───┼───┼───┤
│0,3│1,3│2,3│3,3│4,3│
├───┼───┼───┼───┼───┤
│0,4│1,4│2,4│3,4│4,4│  ← Fila 4 (inferior)
└───┴───┴───┴───┴───┘

Intensidad: 0 (apagado) a 100 (brillo máximo)

Paso 4: Efecto “Latido” (5 minutos)

def latido_corazon():
    """Efecto de corazón latiendo."""
    
    # Corazón completo (brillo 100)
    corazon = [
        [0, 100, 0, 100, 0],
        [100, 100, 100, 100, 100],
        [100, 100, 100, 100, 100],
        [0, 100, 100, 100, 0],
        [0, 0, 100, 0, 0],
    ]
    
    # Corazón apagado (brillo 20)
    corazon_fade = [
        [0, 20, 0, 20, 0],
        [20, 20, 20, 20, 20],
        [20, 20, 20, 20, 20],
        [0, 20, 20, 20, 0],
        [0, 0, 20, 0, 0],
    ]
    
    # Loop de latido
    for _ in range(5):
        hub.display.icon(corazon)
        wait(150)
        hub.display.icon(corazon_fade)
        wait(150)

if __name__ == "__main__":
    latido_corazon()

🛠️ Centro de Diagnóstico

Error Síntoma Causa Solución
“IndexError” Matriz no muestra nada Lista malformada Cada fila debe tener 5 elementos
“LED no enciende” Pantalla vacía Brillo en 0 Verificar valores > 0
“Animacion lenta” Parpadeo wait muy alto Reducir a 100-200ms
“Memory error” IDE se cuelga Frames muy grandes Máximo 25 LEDs por frame

📋 Código de Referencia: Calculadora de Frames

# Función helper para crear matrices automáticamente
def crear_linea_horizontal(fila, brillo=100):
    """Crea una línea horizontal en la fila especificada."""
    matriz = [[0]*5 for _ in range(5)]
    for x in range(5):
        matriz[fila][x] = brillo
    return matriz

def crear_linea_vertical(col, brillo=100):
    """Crea una línea vertical en la columna especificada."""
    matriz = [[0]*5 for _ in range(5)]
    for y in range(5):
        matriz[y][col] = brillo
    return matriz

# Uso:
linea_central = crear_linea_vertical(2)  # Columna del medio
hub.display.icon(linea_central)
