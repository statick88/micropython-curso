Unidad 4: ¿Cómo sabe el robot que hay algo adelante?

⏱️ Tiempo estimada: 50 minutos | 📊 Nivel: Intermedio-Avanzado

Antes de empezar

¿Alguna vez has estado en un pasillo oscuro y vas caminando despacito con las manos adelante? Lo haces porque quieres saber si hay algo antes de chocar.

Eso mismo necesita el robot—una forma de “sentir” si hay algo adelante. Y lo tiene: sensores que le permiten detectar objetos [1][2].

Los sensores: como los ojos del robot

Situación: Estás manejando un carro. Mientras conduces, también estás observando—mirando los otros autos, las señales, los peatones. Si ves algo, reaccinas.

El robot hace lo mismo. Tiene algo que le permite “ver” mientras se mueve. No son ojos exactamente, pero funcionan similar—le dicen qué hay alrededor [2].

┌─────────────────────────────────────┐
│           ROBOT                      │
│                                       │
│    ○○○  ← Este es el "ojo"           │
│                                       │
│    Si hay algo cerca, el sensor      │
│    le dice al robot: "¡Alto, hay     │
│    algo!"                            │
└─────────────────────────────────────┘

El sensor que mide distancia

Situación: ¿Alguna vez has gritado en un canyon o en un lugar vacío? Escuchas el eco. Si el eco vuelve rápido, hay algo cerca. Si tarda mucho, hay algo lejos.

El robot tiene algo que funciona similar—envía un sonido (muy agudo, que no podemos escuchar) y espera el “eco”. Si vuelve rápido, hay algo cerca. Si vuelve lento, hay algo lejos [1][2].

# Primero, traer el sensor
from pybricks.pupdevices import UltrasonicSensor
from pybricks.parameters import Port

# El sensor se conecta al puerto D
sensor = UltrasonicSensor(Port.D)

# Ahora podemos preguntarle: ¿qué tan lejos está?
distancia = sensor.distance()

print(f"Hay algo a {distancia} milímetros")

Practica esto en Thonny:

from pybricks.pupdevices import UltrasonicSensor
from pybricks.parameters import Port
from pybricks.tools import wait

sensor = UltrasonicSensor(Port.D)

# Medir la distancia varias veces
for i in range(5):
    distancia = sensor.distance()
    print(f"Medición {i+1}: {distancia} mm")
    wait(500)

Pero… ¿cómo hace el robot dos cosas a la vez?

Situación: Estás cocinando—mientras revuelves la olla, también estás viendo el reloj para no quemarte. Estás haciendo dos cosas al mismo tiempo.

El robot también puede hacer esto: moverse Y observar al mismo tiempo. No es que sea mágico—simplemente alterna muy rápido entre una cosa y otra, tan rápido que parece que hace todo junto [3].

# En el robot, esto se escribe diferente
# No te preocupes por los detalles ahora—solo observa el patrón

# Una "tarea" que hace algo
async def conducir():
    while True:
        # El robot avanza
        robot.drive(100, 0)  # velocidad 100, sin girar
        wait(50)

# Otra "tarea" que observa
async def observar():
    while True:
        # El robot mide la distancia
        distancia = sensor.distance()
        if distancia < 150:  # Si hay algo a menos de 15cm
            print("¡Hay algo cerca!")
        wait(50)

# Ejecutar ambas al mismo tiempo
run_task(conducir(), observar())

Nota: No necesitas entender todo ahora. Solo recuerda que el robot puede hacer varias cosas “a la vez”—avanzar Y observar al mismo tiempo.

Qué pasa cuando detecta algo

Situación: Estás caminando y de pronto ves un obstáculo—¿qué haces? Te detienes, Quizás haces otro camino.

El robot puede hacer lo mismo [2]:

# Si hay algo a menos de 15 centímetros
if distancia < 150:
    robot.stop()          # Detenerse
    hub.light.on(0)      # Luz roja = cuidado
    robot.turn(180)      # Girar y buscar otro camino
    hub.light.on(5)      # Luz verde = seguir

Es como: 1. Ver el obstáculo → detectar con el sensor 2. “¡Alto!” → stop() 3. Encender luz de alerta → light.on(0) (rojo) 4. Buscar otro camino → turn(180) 5. Seguir adelante → volver a conducir

Hacer que el robot explore solo

Situación: Has visto esos robots de limpieza que van por la casa solos. Van avanzandoy si chocan con algo, giran y siguen.

Vamos a hacer algo similar [2]:

from pybricks.pupdevices import Motor, UltrasonicSensor
from pybricks.parameters import Port
from pybricks.robotics import DriveBase
from pybricks.tools import wait, run_task

# Configurar
motor_izq = Motor(Port.A)
motor_der = Motor(Port.B)
sensor = UltrasonicSensor(Port.D)
robot = DriveBase(motor_izq, motor_der, 56, 112)

# Tarea 1: seguir adelante siempre
async def conducir():
    while True:
        robot.drive(100, 0)  # velocidad 100
        wait(50)

# Tarea 2: observar y reaccionar
async def observar():
    while True:
        distancia = sensor.distance()
        
        if distancia < 150:  # hay algo a menos de 15cm
            robot.stop()
            hub.light.on(0)    # rojo = cuidado
            wait(200)
            robot.turn(180)    # girar 180°
            hub.light.on(5)    # verde = seguir
            wait(500)
        
        wait(50)

# Ejecutar ambas
run_task(conducir(), observar())

¿Qué hace este código? - El robot avanza todo el tiempo - Mientras avanza, el sensor mide la distancia - Si detecta algo a menos de 15cm → se detiene, enciende luz roja, gira 180°, enciende luz verde, y sigue
