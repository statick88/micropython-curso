Laboratorio 4.1: Telemetría Asíncrona con Sensores

⚡ Nivel: Avanzado | ⏱️ Tiempo: 50 minutos

Objetivo del Laboratorio

Implementar un sistema de percepción que funcione en paralelo con la navegación, utilizando programación asíncrona para detectar obstáculos en tiempo real sin detener el robot.

Fundamentos de Ingeniería

El Problema: Sistemas Secuenciales vs Concurrentes

SECUENCIAL (MALO):                    CONCURRENTE (BUENO):
┌─────────────────────────────────┐   ┌─────────────────────────────────┐
│ while True:                      │   │ async def sensar():             │
│     leer_sensor()  ──►  wait     │   │     while True:                 │
│     mover_robot()  ──►  wait     │   │         leer_sensor()           │
│     wait(100)                    │   │         await wait(50)         │
│                                  │   │                                 │
│ Resultado:                       │   │ async def mover():             │
│ Robot CIEGO mientras se mueve   │   │     while True:                │
│                                  │   │         mover_robot()         │
│                                  │   │         await wait(50)         │
│                                  │   │                                 │
│                                  │   │ Resultado:                     │
│                                  │   │ Sensado y movimiento en PARALELO│
└─────────────────────────────────┘   └─────────────────────────────────┘

Paso 3: Sistema de Evitación de Colisiones (15 minutos)

from pybricks.hubs import PrimeHub
from pybricks.pupdevices import Motor, UltrasonicSensor
from pybricks.parameters import Port, Direction, Color
from pybricks.robotics import DriveBase
from pybricks.tools import wait, run_task

hub = PrimeHub()
motor_izq = Motor(Port.A, Direction.COUNTERCLOCKWISE)
motor_der = Motor(Port.B)
sensor_dist = UltrasonicSensor(Port.D)
robot = DriveBase(motor_izq, motor_der, 56, 112)

# Bandera global para comunicación entre tasks
alerta_activa = False

async def vigilante_colisiones():
    """Detecta obstáculos y establece alerta."""
    global alerta_activa
    
    while True:
        distancia = sensor_dist.distance()
        
        if distancia < 150:
            alerta_activa = True
            # Feedback visual de emergencia
            hub.light.on(Color.RED)
            hub.display.text("PARE")
        else:
            alerta_activa = False
            hub.light.off()
        
        await wait(50)

async def explorer_autonomo():
    """Robot que explora hasta encontrar obstáculo."""
    global alerta_activa
    
    robot.use_gyro(True)
    robot.settings(straight_speed=150)
    
    while True:
        # Si hay alerta, detenerse y girar
        if alerta_activa:
            robot.stop()
            
            # Feedback de emergencia
            hub.speaker.beep(frequency=880, duration=200)
            
            # Girar 90 grados y continuar
            await robot.turn(90)
            await wait(500)
        else:
            # Avanzar mientras no haya obstáculos
            await robot.straight(100)
        
        await wait(50)

if __name__ == "__main__":
    run_task(
        explorer_autonomo(),
        vigilante_colisiones()
    )

Paso 4: Integración con Sensores de Color (10 minutos)

from pybricks.pupdevices import Motor, UltrasonicSensor, ColorSensor
from pybricks.parameters import Port, Direction
from pybricks.robotics import DriveBase
from pybricks.tools import wait, run_task

# Agregar sensor de color
sensor_color = ColorSensor(Port.C)  # Puerto C

async def navegacion_bidimensional():
    """
    Navegación que responde a DOS sensores:
    - Ultrasonido: evitar obstáculos
    - Color: seguir líneas
    """
    
    # Sensores activos
    dist = sensor_dist.distance()
    color = sensor_color.color()
    reflexion = sensor_color.reflection()
    
    # Lógica de decisión
    if dist < 150:
        # Obstáculo: girar
        await robot.turn(90)
    elif reflexion < 30:
        # Línea negra detectada: seguirla
        print("📍 Línea detectada")
    else:
        # Espacio libre: avanzar
        await robot.straight(50)
    
    await wait(50)

🛠️ Centro de Diagnóstico

Error Síntoma Causa Solución
“Task no corre” Solo ejecuta 1 función Olvidaste run_task() Usar run_task(a(), b())
“Memory error” IDE se cuelga Sin await wait() en loop Agregar await wait(50)
“Sensor no responde” Distancia = 0 siempre Cable mal conectado Verificar puerto
“RuntimeError” Error en async Función no es async Usar async def
” LED no cambia” Sin respuesta await wait() faltante Verificar loop async
