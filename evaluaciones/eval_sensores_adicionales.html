Pregunta 2 (10 puntos)

¿Cuál es el rango de operación del InfraredSensor y su precisión?

Respuesta esperada:

  • Rango: 50mm a 2000mm (5cm a 200cm)
  • Precisión: ±10mm en近距离, disminuye con distancia
  • Modos: distance(), reflection(), buttons() [1]

Pregunta 3 (10 puntos)

Explica la diferencia entre ColorSensor.color() y ColorSensor.reflection().

Respuesta esperada:

  • color(): Retorna el nombre del color detectado (Color.RED, Color.GREEN, etc.)
  • reflection(): Retorna el porcentaje de reflectividad (0-100)

El modo reflection es más útil para seguimiento de líneas [2].

Parte B: Práctica en Código (50 puntos)

Ejercicio 1: Seguimiento de Línea con Color (25 puntos)

Implementa un programa que siga una línea negra usando ColorSensor:

from pybricks.hubs import PrimeHub
from pybricks.parameters import Port

# Tu código aquí
# ...

Solución esperada:

from pybricks.hubs import PrimeHub
from pybricks.parameters import Port, Stop, Direction
from pybricks.pupdevices import Motor, ColorSensor
from pybricks.tools import wait

hub = PrimeHub()

# Inicializar dispositivos
motor_izq = Motor(Port.B, Direction.COUNTERCLOCKWISE)
motor_der = Motor(Port.A)
sensor = ColorSensor(Port.C)

# Constantes
VELOCIDAD = 200
GIRO = 50
UMBLAL_LINEA = 30  # % reflectividad

# Loop principal
while True:
    # Salir con botón
    if hub.buttons.pressed():
        break
    
    # Leer reflectividad
    reflejo = sensor.reflection()
    
    if reflejo < UMBLAL_LINEA:
        # Línea negra detectada - girar izquierda
        motor_izq.run(VELOCIDAD - GIRO)
        motor_der.run(VELOCIDAD + GIRO)
    else:
        # Fondo claro - girar derecha
        motor_izq.run(VELOCIDAD + GIRO)
        motor_der.run(VELOCIDAD - GIRO)

motor_izq.stop()
motor_der.stop()

Ejercicio 2: Detección de Obstáculos con Infrared (25 puntos)

Crea un sistema que alerte cuando un objeto esté a menos de 100mm:

from pybricks.hubs import PrimeHub
from pybricks.parameters import Port, Color

# Tu código aquí
# ...

Solución esperada:

from pybricks.hubs import PrimeHub
from pybricks.parameters import Port, Color
from pybricks.pupdevices import InfraredSensor
from pybricks.tools import wait

hub = PrimeHub()
sensor = InfraredSensor(Port.C)

UMBRAL_PROXIMIDAD = 100  # mm

while True:
    if hub.buttons.pressed():
        break
    
    distancia = sensor.distance()
    
    if distancia < UMBRAL_PROXIMIDAD:
        # Alerta: objeto cercano
        hub.light.on(Color.RED)
        hub.display.show_icon(HubIcon.EXCLAMATION)
        hub.speaker.beep(frequency=1000, duration=100)
    else:
        # Seguro
        hub.light.on(Color.GREEN)
        hub.display.off()
    
    wait(100)

hub.light.off()

Parte C: Análisis de Datos (20 puntos)

Escenario

Dado el siguiente código de telemetría, analiza los resultados:

from pybricks.hubs import PrimeHub
from pybricks.pupdevices import InfraredSensor, ColorSensor
from pybricks.tools import wait
import json

hub = PrimeHub()
ir = InfraredSensor(Port.C)
color = ColorSensor(Port.D)

datos = []

for i in range(50):
    muestra = {
        "distancia": ir.distance(),
        "reflejo": color.reflection(),
        "color": str(color.color())
    }
    datos.append(muestra)
    wait(100)

# Guardar datos
print(json.dumps(datos))

Preguntas

a) ¿Cuántos segundos dura la captura de datos? (5 puntos)

50 muestras × 100ms = 5000ms = 5 segundos

b) ¿Qué tipo de gráfico usarías para visualizar las distancias? (5 puntos)

Gráfico de línea (line chart) para mostrar tendencia temporal.

c) ¿Cómo exportarías los datos a un archivo CSV en Thonny? (5 puntos)

Usar with open('datos.csv', 'w') as f: y escribir líneas con f.write().

d) ¿Qué mejorarías para una telemetría más precisa? (5 puntos)

  • Reducir el intervalo de muestreo
  • Usar más sensores
  • Implementar filtro de promedio móvil
  • Agregar timestamps con tiempo real
