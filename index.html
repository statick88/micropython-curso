😮 Pero primero… ¿por qué robots?

Imagina que tus estudiantes pueden ver cómo sus instrucciones cobran vida:

Emoji Acción Lo que aprenden
🤖 El robot avanza Secuencias y comandos
💡 La pantalla muestra dibujos Matrices y loops
👁️ Detecta obstáculos Sensores y condicionales
🎵 El robot hace sonidos Eventos y feedback

Eso es exactamente lo que lograremos en este curso.

🛠️ ¿Qué necesitaremos?

Herramienta Para qué Costo
Thonny 🐍 Escribir y probar código Gratis
LEGO Technic Large Hub 🧱 El cerebro del robot (del kit)
Pybricks 🤖 El software del robot Gratis
Un cable USB-C 🔌 Conectar todo (del kit)
Nota💡 ¿Qué es Thonny?

Thonny es un editor de Python gratuito diseñado para quienes están aprendiendo a programar. Fue creado en Estonia y es software libre.

Descarga aquí: https://thonny.org

Tip🚀 ¿Nunca has programado antes?

Si es la primera vez que vas a programar, te recomendamos empezar aquí:

👉 🐍 Módulo de Fundamentos de Python

Antes de trabajar con el hardware, aprende los conceptos básicos de Python con ejemplos relacionados con robótica. Este módulo te dará la base necesaria para aprovechar al máximo el curso.

Tiempo estimado: 10-14 horas (1-2 semanas)

🤖 ¿Qué robot usaremos?

Usaremos el LEGO Technic Large Hub (el cerebro del robot):

Componente Descripción Emojis
Procesador Como el cerebro del robot, controla todo 🧠
4 puertos Para conectar motores y sensores (A, B, C, D) 🔌
Pantalla LED 5x5 puntitos que puedes controlar 💡
Batería Se carga por USB 🔋
Bluetooth También puede funcionar sin cables 📡
Tip💡 Dato interessante

El LEGO Technic Large Hub usa el mismo lenguaje que Python, pero una versión especial llamada MicroPython. Es como si Python pudiera correr dentro de un robot. 🐍🤖

¡Es 20x más pequeño que Python regular pero hace lo mismo!
